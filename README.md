# Прототип информационной медицинской системы

Примерный набор страниц и общий вид функционала для модуля формирования документа приёма врача (ДПВ)

## Setup

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
